# Expense Tracker

## Overview
This project is designed to help users track their expenses efficiently and categorize their spending to understand their financial habits better.

## Features
- User friendly interface
- Add, edit, and delete expenses
- Categorize expenses
- View expense reports and trends

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dhruv-Cmds/Expense-Tracker.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Expense-Tracker
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
